Использование
==============

Основные возможности
--------------------

1. **Показать все раскладки** — просмотр доступных раскладок
2. **Загрузить кастомную раскладку** — добавление пользовательских раскладок из JSON
3. **Анализ комбинаций символов** — полный анализ всех раскладок
4. **Анализ конкретного файла** — анализ текста из указанного файла
5. **Показать результаты** — просмотр результатов последнего анализа

Кастомные раскладки
-------------------
Программа поддерживает загрузку пользовательских раскладок в формате JSON.

Пример формата JSON:

.. code-block:: json

   {
       "name": "Название раскладки",
       "language": "russian/english",
       "layout": [
           ["`", "1", "2", "3", "4", "5", "6", "7", "8", "9", "0", "-", "="],
           ["й", "ц", "у", "к", "е", "н", "г", "ш", "щ", "з", "х", "ъ"],
           ["ф", "ы", "в", "а", "п", "р", "о", "л", "д", "ж", "э"],
           ["я", "ч", "с", "м", "и", "т", "ь", "б", "ю", "."]
       ]
   }
