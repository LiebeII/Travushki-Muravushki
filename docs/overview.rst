Обзор проекта
===============

Цели проекта
------------
Keyboard Layout Analyzer предназначен для:

1. **Объективной оценки** клавиатурных раскладок с использованием научных метрик
2. **Сравнительного анализа** различных раскладок по единым критериям
3. **Помощи в выборе** оптимальной раскладки для конкретных задач
4. **Исследования эргономики** и разработки новых раскладок

Ключевые особенности
--------------------

✨ **Многоязычный анализ**
   - Автоматическое определение языка текста
   - Фильтрация раскладок по языку
   - Поддержка русских и английских раскладок

✨ **Расширенная эргономика**
   - Учет удобства двухсимвольных комбинаций
   - Анализ нагрузки на каждый палец
   - Баланс между левой и правой рукой
   - Расчет штрафов по расстоянию от домашнего ряда

✨ **Поддержка нескольких слоев**
   - Учет второго слоя через Alt
   - Автоматический выбор варианта с наименьшим штрафом
   - Формат "символ1/символ2" для двойных назначений

✨ **Визуализация результатов**
   - Интерактивные графики Matplotlib
   - Сравнительные диаграммы
   - Детальная статистика в табличном виде

✨ **Гибкость и расширяемость**
   - Поддержка пользовательских раскладок через JSON
   - Автоматическая загрузка из папки ready_made_layouts
   - Модульная архитектура для добавления новых критериев

✨ **Подробная статистика**
   - Анализ по словам с учетом контекста
   - Классификация комбинаций по удобству
   - Расчет рейтинга по 4 критериям с весами
