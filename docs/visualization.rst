Визуализация результатов
=========================

Программа генерирует несколько типов графиков с использованием Matplotlib для наглядного представления результатов анализа.

Типы графиков
-------------

1. Статистика нагрузки на пальцы
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- **Назначение:** Показать распределение нагрузки по пальцам
- **Тип:** Столбчатая диаграмма (bar chart)
- **Ось X:** Пальцы (мизинцы, безымянные, средние, указательные)
- **Ось Y:** Количество нажатий (в тысячах или миллионах)
- **Особенность:** Сравнение нескольких раскладок на одном графике

2. Штраф на пальцы по раскладкам
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- **Назначение:** Сравнить общий штраф для разных раскладок
- **Тип:** Столбчатая диаграмма с цветовым кодированием
- **Ось X:** Раскладки
- **Ось Y:** Общий штраф (чем меньше, тем лучше)
- **Цвет:** Градация от зеленого (низкий штраф) к красному (высокий штраф)

3. Баланс между руками
~~~~~~~~~~~~~~~~~~~~~~~
- **Назначение:** Показать распределение нагрузки между руками
- **Тип:** Горизонтальная столбчатая диаграмма
- **Ось Y:** Раскладки
- **Ось X:** Процент нагрузки (0-100%)
- **Область идеала:** Выделенная зона 45-55%
- **Индикаторы:** Стрелки для левой и правой руки

4. Распределение комбинаций по удобству
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- **Назначение:** Показать соотношение типов комбинаций
- **Тип:** Сгруппированная столбчатая диаграмма
- **Группы:** Раскладки
- **Столбцы:** Удобные, Частично удобные, Неудобные
- **Высота столбца:** Процент от общего числа комбинаций

Настройка визуализации
----------------------

### Параметры отображения:
- **Размер фигуры:** Адаптируется под количество раскладок
- **Цветовая схема:** Используется набор цветов для лучшего восприятия
- **Подписи:** Автоматическое форматирование больших чисел (K, M)
- **Легенда:** Автоматическое размещение

### Сохранение графиков:
Графики автоматически сохраняются в файлы:
- `finger_load_comparison.png` — нагрузка на пальцы
- `total_penalty_comparison.png` — штрафы
- `hand_balance_comparison.png` — баланс рук
- `combo_type_distribution.png` — типы комбинаций

Интерпретация графиков
----------------------

### На что обращать внимание:

1. **Нагрузка на пальцы:**
   - Идеал: Равномерное распределение
   - Проблема: Перегрузка указательных пальцев
   - Решение: Перераспределение частых символов

2. **Общий штраф:**
   - Чем ниже столбец, тем лучше
   - Учитывайте абсолютные значения
   - Сравнивайте с другими раскладками

3. **Баланс рук:**
   - Идеал: Столбец в зоне 45-55%
   - Красная зона: Сильный перекос (>70% или <30%)
   - Желтая зона: Умеренный перекос

4. **Типы комбинаций:**
   - Цель: Максимум синих столбцов (удобные)
   - Минимизация красных столбцов (неудобные)
   - Желтые столбцы (частично удобные) — нейтральны

Использование в отчетах
------------------------

Графики можно использовать для:
- **Сравнительного анализа** разных раскладок
- **Обоснования выбора** конкретной раскладки
- **Научных исследований** эргономики
- **Демонстрации** преимуществ альтернативных раскладок

Советы по анализу:
1. **Смотрите на общую картину:** Не зацикливайтесь на одном графике
2. **Учитывайте веса критериев:** Удобство комбинаций важнее баланса рук
3. **Проверяйте консистентность:** Результаты на разных текстах должны быть похожи
4. **Учитывайте субъективные факторы:** Графики не учитывают привычки пользователя
