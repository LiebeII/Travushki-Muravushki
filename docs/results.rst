Результаты анализа "Война и мир" Л.Н. Толстого
----------------------------------------------

ЙЦУКЕН:
   **Преимущества:**
   - ✓ Низкий общий штраф (4.9M vs среднее 5.2M)
   - ✓ Отличный баланс между руками (96.9% vs среднее 87.9%)

   **Недостатки:**
   - ✗ Низкий процент удобных комбинаций (19.9% vs среднее 22.9%)
   - ✗ Высокая нагрузка на указательные пальцы (1.8M vs среднее 1.2M)

Скоропись:
   **Преимущества:**
   - ✓ Низкий общий штраф (4.8M vs среднее 5.2M)

   **Недостатки:**
   - ✗ Низкий процент удобных комбинаций (17.6% vs среднее 22.9%)
   - ✗ Плохой баланс между руками (77.9% vs среднее 87.9%)

   **Преимущества:**

   **Недостатки:**

   - ✓ Низкая нагрузка на указательные пальцы (697k vs среднее 1.2M)

   **Недостатки:**
   - ✗ Высокий общий штраф (5.5M vs среднее 5.2M)
   - ✗ Плохой баланс между руками (83.2% vs среднее 87.9%)

Визуализация
------------
Программа генерирует несколько типов графиков:

1. **Нагрузка на пальцы** — столбчатая диаграмма распределения кликов по пальцам
2. **Общий штраф** — сравнение штрафов для разных раскладок
3. **Распределение штрафов между руками** — круговые диаграммы для каждой раскладки
4. **Одноручные комбинации** — соотношение удобных/неудобных комбинаций
5. **Распределение комбинаций по длинам** — детальный анализ по длине комбинаций
