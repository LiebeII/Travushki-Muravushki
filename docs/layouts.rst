Поддерживаемые раскладки
=========================

Русские раскладки
------------------
- **ЙЦУКЕН** — стандартная русская раскладка
- **Скоропись** — эргономичная русская раскладка
- **Фонетическая (яВерт)** — раскладка, где русские буквы соответствуют английским по звучанию
- **Вызов** — альтернативная русская раскладка

Английские раскладки
---------------------
- **QWERTY** — стандартная английская раскладка
- **Dvorak** — эргономичная раскладка, разработанная для скорости и удобства
- **Colemak** — современная эргономичная раскладка
- **Workman** — раскладка, оптимизированная для английского языка
